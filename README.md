# Phone Cobalt Checker

Simple standalone HTML app for the poster QR. It is ready for GitHub Pages.

Open `index.html` in a browser. Visitors can:

- search 650+ common Pakistan-market phone models
- see estimated cobalt, DRC-linked cobalt exposure, battery CO2e, and labour-risk pressure
- scale the selected phone to a whole classroom of phones
- share/copy a personal result
- use battery capacity if their model is missing
- see a realistic phone-and-minerals hero image with an exhibit-style interface

This folder is standalone and can be uploaded anywhere.

For GitHub Pages steps, see `PUBLISH_GITHUB_PAGES.md`.

## Estimate note

Phone brands rarely publish the exact cobalt mass in each battery. This calculator uses public battery capacity and a simple cobalt-content range for cobalt-containing lithium-ion batteries. It is an educational estimate, not a supply-chain audit.

The Congo and labour-risk numbers are proxy estimates for a poster exhibit. They combine cobalt mass with DRC global mine-share context, a broad artisanal-mining share, and a transparent worker-minute assumption. They do not prove that a specific phone brand or model used child labour.
